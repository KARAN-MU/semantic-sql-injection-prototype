\# Semantic SQL Injection Prototype



This repository contains a learning prototype demonstrating

semantic SQL injection detection using an NLP/ML model.



\## Approach

\- Uses Sentence-BERT to generate semantic embeddings

\- Compares SQL queries by intent using cosine similarity

\- Detects syntactically different but semantically equivalent SQLi variants



\## Files

\- prototype.py: baseline semantic prototype

\- improved\_prototype.py: hybrid semantic + pattern-based version

\- samples.py: test data



\## Note

This is a learning prototype, not production-ready code.
